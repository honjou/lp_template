# イントロ

ランディングページのテンプレートです。

ランディングページの王道パターンをサンプル画像としていれていますので、クライアントとの最初のたたき台としても利用できるかと思います。

多くのテンプレートではお問い合わせ機能はついていませんが、このテンプレートではLaravelで作成したメールフォームを実装しています。


## レスポンシブ

575.98pxをブレイクポイントにし、PC用の画像とスマホ用の画像に切り替わります。

PC・スマホ用の画像を差し替えるだけで、ランディングページが出来上がります。


## Bootstrap4

Bootstrap4をベースにしています。

カスタマイズに関してはBootstrap4のコンポーネントルールをご参照ください。

[Bootstrap4 Cheat Sheet](https://hackerthemes.com/bootstrap-cheatsheet/)

また、アイコンに関しては Font Awesome5 を使用しています。


## Laravelのメール機能を実装

HTML5のバリデーションだけだと心配なので、Laravelでのバリデーションも実装しています。

* HTMLバリデーション
* Laravelバリデーション(フォームリクエスト)
